## GUIDE QUESTIONS ANSWERS (Simple Explanation)
# 1. Weakest-performing classes?

Classes with many wrong predictions in confusion matrix.

# 2. Precision, Recall, F1 variation?

Some classes performed better because their features were easier to recognize.

# 3. Low recall means?

The model missed many actual images of that class.

# 4. AUC vs Accuracy?

Accuracy checks overall correctness.
AUC measures how well classes are separated.

# 5. Effect of data augmentation?

Improved validation accuracy and reduced overfitting.

# 6. Why Batch Normalization?

It stabilizes training and speeds learning.

# 7. Role of Dropout?

Prevents memorization by randomly disabling neurons.

# 8. Early Stopping?

Stops training before overfitting happens.

# 9. Improvements observed?

Higher accuracy, lower loss, better F1-score and AUC.

# 10. Most useful enhancement?

Usually Data Augmentation + Dropout because they improve generalization.
